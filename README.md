# FormForge

Application to create forms based on imported templates or starting from scratch

## Instructions to Run the project

- Clone the repository, then run `npm install` in the terminal to get all of the necessary dependencies for the application.
- To start the application running, execute `npm run dev` in the terminal, and the application should start running on port 5173

## Tools/Libraries used

I made use of npm to handle dependencies for the application, as well as the Bootstrap framework for some of the styling such as the buttons and the header bar.

## External API Endpoint

I used the todos endpoint JSON Placeholder. The todos were grouped into sets of five, and then their descriptions were used as the question's text. I added to possibility for multiple choice questions to be generated randomly with empty choices fields.
